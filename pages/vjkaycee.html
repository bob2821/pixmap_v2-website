KayCee has been behind the lens and behind the patch for a generation of Indian stages. From the first Sunburn Goa LED back-wall to touring rigs for NH7 Weekender, Supersonic, and a dozen underground warehouse nights nobody's supposed to talk about.

He built PIXMAP because the existing tools made him feel like a technician, not an artist. It ships with the opinions of someone who has gaffed XLR in a downpour at 3am and still had to deliver a showday build before doors.

"Every pixel matters. The tools we had never understood that. So I built one that does."

- VJ KayCee, Founder · Pixadoo
